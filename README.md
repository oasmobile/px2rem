## Install
```
npm install @oasis-games/px2rem --save-dev
```
## How to use px2rem?
```
/* base.js */  

import px2rem from '@oasis-games/px2rem';  

px2rem(640);  

/* *.scss */  

@import "~@oasis-games/px2rem/rem.scss";  

width:rem(640px);
```
